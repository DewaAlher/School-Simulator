School Simulator

A procedural school life simulation game where every NPC has their own personality, memories, needs, relationships, goals, and daily routines.

This project aims to create a living school world that continues to evolve even without player intervention. Every character should feel unique, make decisions independently, and create emergent stories through their interactions.

Vision

Create a school simulation where players don't just play the game—they observe, influence, and become part of a living world.

Current Status
